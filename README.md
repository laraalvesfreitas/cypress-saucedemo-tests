# Swag Labs - Testes End-to-End

Projeto de automação de testes End-to-End desenvolvido com Cypress utilizando a aplicação Swag Labs como sistema sob teste.

O objetivo deste projeto é praticar conceitos de automação de testes, validando os principais fluxos da aplicação e aplicando boas práticas de QA e automação.

> ⚠️ Projeto em desenvolvimento. Novos cenários de teste, melhorias e refatorações serão adicionados gradualmente por meio de novos commits.

## Tecnologias Utilizadas

- Cypress
- JavaScript
- Node.js
- Git
- GitHub

## Cenários de Teste

- Login
- Validação de mensagens de erro
- Listagem de produtos
- Adição e remoção de produtos do carrinho
- Checkout
- Fluxo completo de compra

## Como Executar o Projeto

### Instalar as dependências

```bash
npm install
```

### Executar os testes em modo interativo

```bash
npx cypress open
```

### Executar os testes em modo headless

```bash
npx cypress run
```

## Objetivo

Este projeto faz parte dos meus estudos em automação de testes, com foco no desenvolvimento de habilidades práticas utilizando Cypress para testes End-to-End.
